# bench

Using a [panda](https://comma.ai/shop/products/panda) from comma.ai, we can simulate the CAN messages used to interact with different automotive infotainment systems.

## Systems

- Ford Sync 3 infotainment unit (APIM)
- Audi/VW/Skoda MIB
